# My Little Planet

An interactive 3D ecology learning experience. Restore a damaged planet using rain, sunlight, plants, and care!

## Features

- 🎮 Interactive 3D planet visualization
- ✋ Hand gesture control via webcam
- 🎤 Voice commands support
- 🔄 Real-time ecosystem simulation
- 🦋 Endangered species discovery

## Getting Started

```bash
npm install
npm run dev
```

## Tech Stack

- React + TypeScript
- Three.js + React Three Fiber
- MediaPipe Hands for gesture recognition
- Web Speech API for voice commands
- Tailwind CSS
